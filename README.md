# my-tic-tac-toe-ReactJS

Tic tac toe with move calculations for winning and avoid loosing. Using: ReactJs hooks, Typescript, simple automation tests.

  <i><b>-Green</b> square indicator for winning move</i>
  </p>

  </p>

 `npm start`
